 # VR-Space_Sweeper
### Unity VR Project
VR 기기를 움직여 무작위로 날아오는 유성체를 피하고 구글 카드보드의 자석 스위치를 통해 우주잔해를 청소하여 우주선을 업그레이드하는 생존 게임입니다.
